## Ecommerce User

### Ecommerce User Show (GET)

```
ecommerce-users/:id
```

### Ecommerce User Update (PATCH)

```
ecommerce-users/:id
```

| Arguments     | Type     | Description  |
| :------------ | :------- | :----------- |
| `name`        | `string` | **Nullable** |
| `phone`       | `string` | **Nullable** |
| `email`       | `string` | **Nullable** |
| `dateOfBirth` | `Date`   | **Nullable** |
